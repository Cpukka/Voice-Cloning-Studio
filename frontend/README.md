# Voice Cloning Studio - Production Ready AI Platform

## 🎯 Features

- **User Authentication**: JWT-based auth with role management (admin/user)
- **Voice Cloning**: Upload 3-30s audio samples to create custom voice models
- **Text-to-Speech**: Generate natural speech using cloned voices
- **API Access**: RESTful API with API key authentication for external integration
- **Modern UI**: Dark mode dashboard with responsive design
- **Audio Management**: Download and manage generated audio files
